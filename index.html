Flex Frog

Welcome to Flexbox Froggy, a game where you help Froggy and friend by writing CSS code! Guide this frog to the lilypad on the right by using the justify-content property, which aligns items horizontally and accepts the following values:

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around

For exaple, justify-content: flex-end; will move the frog to the right.

#pond{
display:flex;
}
